### 📝 기능 목록

#### 입력
- 로또 구입 금액을 입력받는 기능
- - [예외] 구입 금액은 숫자여야 한다.
- - [예외] 구입 금액은 0이 아닌 양수여야 한다.
- - [예외] 구입 금액은 1000으로 나누어 떨어져야 한다.


- 당첨 번호를 입력받는 기능
- - [예외] 당첨 번호는 각각이 숫자여야 한다.
- - [예외] 당첨 번호는 중복되서는 안된다.
- - [예외] 당첨 번호는 각각이 1 ~ 45의 수여야 한다.


- 보너스 번호를 입력받는 기능
- - [예외] 보너스 번호는 숫자여야 한다.
- - [예외] 보너스 번호는 당첨 번호와 중복되서는 안된다.
- - [예외] 보너스 번호는 1 ~ 45 까지의 수여야 한다.

#### 출력
- 구입한 로또 개수를 출력하는 기능
- 구입한 로또들을 출력하는 기능 
- 당첨 통계를 출력하는 기능

#### 도메인
- 돈으로 로또를 구매하는 기능
- 랜덤하게 로또를 생성하는 기능 


- 로또 정보를 저장하는 기능
- 당첨 번호를 저장하는 기능
- 보너스 번호를 저장하는 기능


- 당첨 번호와 로또를 비교해 통계를 만들어내는 기능
- 수익률을 계산하는 기능
